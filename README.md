# Classical and Modern Artificial Intelligence

## Overview
This repository contains notes, experiments, and exercises developed during the first unit of the Master's program in Artificial Intelligence Applied. The focus of this module is understanding the evolution of Artificial Intelligence from classical approaches to modern AI techniques.

## Topics Covered
- Introduction to Artificial Intelligence
- Classical AI approaches
- Machine Learning fundamentals
- Generative AI
- AI ethics and responsible AI
- Real-world AI use cases
- Artificial Intelligence Software Platforms

## Technologies Used
- Python
- Jupyter Notebook
- Data analysis libraries

## Purpose
The goal of this repository is to document the learning process and implement small experiments to understand the foundations of Artificial Intelligence.

## Key Learnings

- Understanding the difference between classical AI and modern AI
- Introduction to machine learning concepts
- Overview of generative AI systems
- Ethical considerations in AI development
